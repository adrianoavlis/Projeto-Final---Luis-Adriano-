Plataforma de Vendas Informais

Plataforma

O projeto visa atender a uma demanda latente de uma comunidade específica, oferecendo uma solução inovadora que não apenas facilitará o comércio online local, mas também proporcionará insights valiosos sobre o relacionamento entre pessoas e produtos.

Funcionalidades

  • Cadastro de Lojistas: Permite que os vendedores locais se cadastrem na plataforma, fornecendo informações sobre seus produtos e serviços.
  • Catálogo Online: Os produtos dos lojistas cadastrados serão disponibilizados em um catálogo online acessível ao público, facilitando a visualização e a busca pelos consumidores.
  • Análise de Dados: O sistema realizará análises de dados para identificar padrões de comportamento de compra, incluindo quais produtos são mais vendidos, histórico de preços, preferências por faixa etária e nicho de mercado.
  • Registro de Preços: Manterá um histórico dos preços dos produtos ao longo do tempo, permitindo aos consumidores e lojistas acompanhar as variações de preço.
  • Personalização de Nicho de Mercado: Oferecerá dados sobre produtos mais vendidos em nichos de mercado específicos, permitindo uma abordagem mais personalizada para os consumidores e lojistas.
  • Previsão de Compra: Utilizará dados históricos e padrões de compra para fazer previsões sobre a demanda futura de produtos, auxiliando os lojistas no planejamento de estoque e produção.
  • Extração de Dados: Realizará a extração de informações de outras plataformas de marketplace por meio de técnicas de scrapping, garantindo um amplo acesso a dados relevantes para análise.

Tecnologias Utilizadas

  • Backend: Java EE 7, GlassFish
  • Banco de Dados: SQL, MySQL
  • Frontend: React ou React Native (definindo), JSP, HTML

Documentos Principais

  • Relatório do TCC: Documento detalhando todas as etapas, resultados e conclusões do trabalho.
  • Formulário de Pesquisa: Formulário utilizado para coletar dados relevantes ao projeto.
  • Objetos de Scrapping: Artefatos produzidos para a obtenção de dados de produtos na web.
  • Documentação de Software: Documentação da análise e desenvlvimento da plataforma.
